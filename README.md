# Medical-Imaging-AI-Capstone

## Project Overview
This project focuses on building a convolutional neural network pipeline for chest X-ray image classification. The goal is to load and preprocess X-ray images, train a CNN-based model, evaluate its performance and eventually use explainability tools to understand what regions of the image influenced the prediction.
